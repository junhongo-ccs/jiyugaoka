# 自由が丘再開発マップ PoC レポート

## 概要

この PoC は、自由が丘駅周辺の低層商業地、新築地点、道路整備、駅前再開発を同じ座標系で扱い、2023 年から 2026 年への変化を時系列で見せることを目的としている。

## 読み取れる変化

- 2023 年時点では「旧低層雑貨街」が街の記憶を代表する地点として存在する
- 2025 年には「奥沢7丁目新築戸建」が現れ、住宅価格上昇の兆しを示す
- 2026 年には「自由が丘駅前タワー」「自由が丘1丁目29番地区」「楽歩地区道路拡幅」が重なり、駅前の景観と導線が同時に再編される

## 空間的な見どころ

- 2023 年の低層商業地点と 2026 年の再開発予定地点は近接しており、街の記憶と新しい開発圧が同じ駅前圏内で重なっている
- 道路拡幅地点を別ステータスとして持つことで、建物だけでなく歩行環境の変化も同一マップ上で読める

## 価格データの扱い

- `price_up` は PoC 用の補助属性であり、現時点では新築地点にのみ設定している
- 今後は出典付きの価格系列データに差し替えることで、ヒートマップや経年比較に拡張できる

## 次の発展

- Polygon や LineString に拡張し、実際の街区形状を扱う
- QGIS で出力した PNG や GIF を GitHub Pages 側に組み込み、閲覧者が地図とレポートを一体で読めるようにする
- CODEX CLI に SQL 結果を渡して、毎回の更新時に文章を自動生成する
